Smetai — AI Platform for Commercial Proposals from PDF Specifications

Project Goal

To automate the creation of commercial proposals (CPs) based on engineering PDFs, 
specifications, and technical drawings using generative AI models like 
Claude Code and Gemini Vision/Pro/CLI.

Our main focus is industrial and construction companies that work with complex 
technical documentation.

Smetai 5.0 Pipeline (MVP)

1. Input
   - Upload PDF files (drawings, specifications)
   - Interface via Telegram bot or web panel

2. PDF Processing
   - Convert PDF to high-resolution PNG
   - Gemini Vision identifies tables, drawings, and text blocks

3. OCR + Correction
   - Azure OCR (or GCP Vision AI) extracts HTML/text from images
   - Claude or Gemini Pro structures the data and applies a prompt-based 
     correction layer

4. Data Normalization & Enrichment
   - Clean up noisy or incomplete data
   - Match against reference catalogs
   - Evaluate chunk quality and re-prompt if needed

5. Price Matching
   - Manual or automatic extraction from supplier sources
   - Gemini Vision reads from screenshots or online price sheets

6. Proposal Generation
   - Claude Code/Pro generates the final CP with structure, positions, and pricing
   - Outputs PDF or in-chat message for quick delivery

Scaling Potential

Technically:
   - Modular AI agents (FastAPI microservices)
   - Accessible via API, bots, and web dashboards
   - Supports streaming, batch, and parallel execution
   - Ready for cloud deployment (Cloud Run, GKE)

By Industry:
   - Compatible with various sectors: construction, manufacturing, food processing, etc.
   - Custom templates by country and tender requirements
   - Supplier optimization and logistics module in roadmap
